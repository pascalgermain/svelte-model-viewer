# Svelte Model Viewer

> A Svelte Model viewer, made with SvelteKit.

> https://pascalgermain.github.io/svelte-model-viewer/

## Project setup

```bash
pnpm install
```

### Serves and hot-reloads for development

```bash
pnpm run dev
```

### Builds as static site for production

```bash
pnpm run build
```
